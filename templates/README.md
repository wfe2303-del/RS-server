`xlsx` 파일은 바로 Excel에서 열 수 있는 템플릿이고, `csv` 파일은 내용 확인이나 대량 복사 붙여넣기용입니다.

사용 순서:

1. `광고RS정산_템플릿.xlsx`를 열어 실제 데이터로 바꿉니다.
2. 구글드라이브에 업로드해서 구글시트로 변환합니다.
3. 시트 탭 이름을 `광고RS정산`으로 맞추거나, [apps-script/Code.gs](/C:/Users/user/Desktop/classaroundRS-main/apps-script/Code.gs:1)의 `sheetName`을 바꿉니다.
4. 구글시트 ID를 [apps-script/Code.gs](/C:/Users/user/Desktop/classaroundRS-main/apps-script/Code.gs:1)의 `spreadsheetId`에 넣습니다.
